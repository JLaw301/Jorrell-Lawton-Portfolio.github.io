Video Game Idea Generator

  • Info

This project is a video game idea generator application made in python using the Tkinter library. When you click the generate button, the app randomly generates a new genre, theme, and mechanic for a potential game idea. If you like a specific topic that was generated, you can press the lock button next to it so the topic wont be changed. Once you've generated an idea that you like, you can click the favorite button to save your idea in a text file.
